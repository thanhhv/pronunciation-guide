# Exercises Expansion Plan

## Context
The `/exercises` page currently has 4 exercise sets. This plan expands to **16 total** (12 new) across 4 phases, covering all major vowel and consonant contrasts that Vietnamese learners struggle with. Some exercises use 3–4 target sounds (all options are targets, no distractors) for harder discrimination tasks.
